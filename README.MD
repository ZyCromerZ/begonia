# INFO kernel
* QuantumKiller-L : 
    * Undervolted cpu a little
    * 806Mhz Gpu clock 
* QuantumKiller-N :
    * normal CPU
    * 850Mhz Gpu clock 
* Tercydux :
    * normal CPU
    * 1000Mhz Gpu clock
* Tercydux-OC : 
    * overclock CPU BIG cores to 2.5Ghz
    * 1510Mhz Gpu clock
